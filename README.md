# shaka-hls-dash-player-demo

Live demo and showcase site for [shaka-hls-dash-player](https://github.com/RoncoJhon/shaka-hls-dash-player).

**Live site:** [roncojhon.github.io/shaka-hls-dash-player-demo](https://roncojhon.github.io/shaka-hls-dash-player-demo/)

## What is this?

This repo hosts the GitHub Pages demo site for the `shaka-hls-dash-player` web component. It shows:

- A live embedded player with a real HLS/DASH stream
- Quick start code snippet
- Feature overview
- Two embed methods (CDN vs self-hosted)
- Browser compatibility table
- Keyboard shortcuts
- Expected folder structure

## The actual component

The web component source code lives in a separate repo:

- **Repo:** [github.com/RoncoJhon/shaka-hls-dash-player](https://github.com/RoncoJhon/shaka-hls-dash-player)
- **CDN:** `https://cdn.jsdelivr.net/gh/RoncoJhon/shaka-hls-dash-player/shaka-hls-dash-player.js`
- **Docs:** See the README in the main repo for full API reference

## About

Built by [RoncoDev](https://roncodev.com). This player ships with [DASH/HLS Packager](https://roncodev.com) — a desktop app that converts any video into multi-resolution HLS & DASH. See it on the [product demo page](https://roncodev.com/demo).
